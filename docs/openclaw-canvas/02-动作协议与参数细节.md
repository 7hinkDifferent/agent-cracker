# 02. 动作协议与参数细节

基准实现：`projects/openclaw/src/agents/tools/canvas-tool.ts:18`

## 1) action 枚举

- `present`
- `hide`
- `navigate`
- `eval`
- `snapshot`
- `a2ui_push`
- `a2ui_reset`

## 2) 通用参数

- `gatewayUrl`, `gatewayToken`, `timeoutMs`
- `node`（可为空，工具会尝试默认节点）

节点解析：`projects/openclaw/src/agents/tools/nodes-utils.ts:58`

## 3) 各动作参数/行为

### `present`

- 入参：`target`/`url`（二选一或都无），可带 `x/y/width/height`。
- 发送命令：`canvas.present`。
- 细节：工具里兼容 `target` 和 `url` 两种字段名。

位置参数组装：`projects/openclaw/src/agents/tools/canvas-tool.ts:108`

### `hide`

- 发送命令：`canvas.hide`
- 返回：`{ ok: true }`

### `navigate`

- 入参：`url`（支持 `target` 作为别名）
- 发送命令：`canvas.navigate`

### `eval`

- 入参：`javaScript`
- 发送命令：`canvas.eval`
- 返回：优先读取 `payload.result` 文本。

### `snapshot`

- 入参：`outputFormat`(`png/jpg/jpeg`)、`maxWidth`、`quality`
- 发送命令：`canvas.snapshot`
- 节点返回 payload 需满足：`{ format, base64 }`
- tool 会落临时文件并生成图片结果。

解析器：`projects/openclaw/src/cli/nodes-canvas.ts:9`

### `a2ui_push`

- 入参：`jsonl` 或 `jsonlPath`（必须有一个）
- 发送命令：`canvas.a2ui.pushJSONL`
- `jsonlPath` 会做本地路径白名单校验，避免越权读取。

路径校验：`projects/openclaw/src/agents/tools/canvas-tool.ts:29`

### `a2ui_reset`

- 发送命令：`canvas.a2ui.reset`

## 4) CLI 行为与 tool 行为对齐点

CLI 实现：`projects/openclaw/src/cli/nodes-cli/register.canvas.ts:79`

- CLI 和 tool 均以 `node.invoke(canvas.xxx)` 作为统一入口。
- CLI 的 `a2ui push --text` 会先构造 v0.8 JSONL。
- 若检测到 v0.9，会明确报错（当前仅支持 v0.8）。

JSONL 校验器：`projects/openclaw/src/cli/nodes-cli/a2ui-jsonl.ts:31`
