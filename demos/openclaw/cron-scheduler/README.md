# OpenClaw — Cron Scheduler

复现 OpenClaw 的定时调度系统（Dimension 11: 安全模型与自治）。

## 机制说明

OpenClaw 支持 3 种调度类型，让 Agent 可以自主执行定时任务。

| 类型 | 格式 | 说明 |
|------|------|------|
| `at` | `at:1234567890` | 绝对时间点（一次性） |
| `every` | `every:30m` | 相对间隔（循环） |
| `cron` | `cron:*/5 * * * *` | Cron 表达式（循环） |

### 核心特性

```
Heartbeat 循环（每分钟一次）
  │
  ├─ 检查到期任务 → 执行（最多 3 个并发）
  │   ├─ 成功 → 计算下次执行时间
  │   └─ 失败 → 指数退避: 30s → 1m → 5m → 15m → 60m
  │
  ├─ 检测遗漏任务（宕机恢复后补执行）
  │
  └─ 返回 HEARTBEAT_OK（空闲）或任务列表
```

### 错误退避

| 连续失败次数 | 退避时间 |
|------------|----------|
| 1 | 30s |
| 2 | 1min |
| 3 | 5min |
| 4 | 15min |
| 5+ | 60min |
